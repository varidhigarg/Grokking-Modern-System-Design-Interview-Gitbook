# Grokking Modern System Design Interview

Use this page as the course index. Read the sections in order, then use the
previous/next links at the top and bottom of every lesson to continue on mobile.

This local copy follows the available files in this repository. The current
Educative course page lists additional newer modules after Google Docs, but
those lessons are not present in this repo yet.

## How to Read This

1. Start with the interview process and foundational concepts.
2. Move through building blocks in order. These are the reusable parts behind
   most system design answers.
3. Finish with the case studies. They apply the building blocks to real product
   designs.
4. Read the failures and conclusions last.
